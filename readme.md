# How it works
```
Simple program show  carrier, region and  timezone of phone number.

```
# How to use

### Python3+ Linux
```
sudo apt update

sudo apt upgrade

sudo apt install git

sudo apt install python3-pip

sudo apt install python

git clone https://github.com/CylindricAnt606/Phone-Number-Checker.git

cd phone-number-checker

pip (or pip3) install -r requirements.txt
-------------------------------------------------------------------------------------|
IF THE ABOVE OPTION DOESN'T WORK THEN USE THESE IF NOT THEN SKIP TO NEXT STEP        |
                                                                                     |
pip (or pip3) install pysimplegui                                                    |
                                                                                     |
pip (or pip3) install phonenumbers                                                   |
                                                                                     |
pip (or pip3) install pyperclip                                                      |
-------------------------------------------------------------------------------------|

python phone.py
```

### Windows
```
GO TO RELEASES AND DOWNLOAD THE Phone-Checker.zip AND THEN EXTRACT IT ONCE EXTRACTED GO INSIDE AND CLICK ON DIST ONCE DONE CLICK ON phone.exe

```
