# How it works
```
Simple program show  carrier, region and  timezone of phone number.

```
# How to use

### Python3+
```
git clone https://github.com/adrijano/phone-number-info.git

cd phone-number-info && pip install -r requirements.txt

python phone.py
```

### Windows
```
cd dist

phone.exe
```
